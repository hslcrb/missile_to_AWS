# 📓 프로젝트 변경 이력 (Changelog)

이 프로젝트의 주요 마일스톤 및 업데이트 내역을 기록합니다.

---

## [Gamma] - 2026-03-19 (현재)
### 보안 및 안정성 패치
- **DPAPI 암호화 도입**: AWS 자격 증명 저장 시 Windows 데이터 보호 API를 사용하여 보안을 대폭 강화했습니다.
- **크래시 버그 수정**: 설정 창 닫기 및 비동기 저장 시 발생하던 `Access Violation` 크래시를 완벽히 해결했습니다.
- **리소스 누수 방지**: GDI 객체(폰트, 브러시) 및 스레드 핸들 누수를 전수 조사하여 수정했습니다.
- **JSON 이스케이프**: 설정 파일 생성 시 특수 문자 처리 로직을 추가하여 데이터 무결성을 확보했습니다.
- **문서화**: 라이선스(MIT), 리드미(README), 사용법(USAGE), 기여 가이드(CONTRIBUTING)를 전면 개편했습니다.

## [Beta] - 2026-03-17
### UI/UX 개선 및 인터랙티브 요소
- **인터랙티브 로고**: 최상단 로고에 마우스 오버 툴팁, 클릭 시 GitHub 이동, 우클릭 링크 복사 기능을 추가했습니다.
- **기능 제안/AWS 콘솔 링크**: 상단에 관련 링크로 즉시 이동 가능한 커스텀 버튼 2개를 추가했습니다.
- **설정 팝업 고도화**: 리소스 즐겨찾기 관리 및 설정 파일 상태 조회 기능이 포함된 대화상자를 구현했습니다.

## [Alpha] - 2026-03-15
### 최초 릴리스
- **프로세스 할로잉 엔진**: `aws-nuke.exe` 인메모리 주입 및 실행 기능 구현.
- **Native GUI**: 순수 Win32 API 기반의 고속 GUI 개발.
- **퍼지 검색 및 한글화**: 400여 종의 AWS 리소스 한글화 및 초성 검색 엔진 탑재.
- **실시간 터미널**: 백그라운드 프로세스 입출력 파이핑 기능 구현.
